# Порты сервисов

## Development (`docker-compose.yml`)

| Сервис       | Порт      | Описание                     |
|-------------|-----------|------------------------------|
| API         | 8000      | FastAPI backend              |
| Frontend    | 5173      | Vite dev server              |
| PostgreSQL  | 5432      | База данных                  |
| Redis       | 6379      | Кэш / очереди               |
| MinIO API   | 9000      | S3-совместимое хранилище     |
| MinIO UI    | 9001      | Консоль управления MinIO     |

## Production (`docker-compose.prod.yml`)

| Сервис       | Порт      | Описание                     |
|-------------|-----------|------------------------------|
| Nginx       | 80        | HTTP → редирект на HTTPS     |
| Nginx       | 443       | HTTPS (SSL/TLS)              |
| RabbitMQ UI | 15672     | Консоль управления RabbitMQ  |
| Prometheus  | 9090      | Метрики                      |
| Grafana     | 3000      | Дашборды мониторинга         |

### Внутренние порты (не проброшены наружу в prod)

| Сервис       | Порт      | Описание                     |
|-------------|-----------|------------------------------|
| API         | 8000      | Доступен через nginx          |
| Frontend    | 80        | Статика через nginx           |
| PostgreSQL  | 5432      | Только внутри docker-сети     |
| Redis       | 6379      | Только внутри docker-сети     |
| MinIO       | 9000      | Только внутри docker-сети     |
| RabbitMQ    | 5672      | AMQP, только внутри сети      |
