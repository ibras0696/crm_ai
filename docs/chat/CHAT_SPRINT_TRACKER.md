## Цель
Сделать доработки мессенджера без изменения текущего визуального стиля и UX-паттернов.

## Объем Работ
- Добавить удаление чатов и групп.
- Добавить открытие профиля пользователя по клику на аватар.
- Добавить строгую проверку прав администратора группы на управление чатом.
- Добавить вход в карточку группы.

## Визуальные Ограничения
- Сохраняем текущий стиль интерфейса (темная тема, карточки, отступы, типографика, поведение контролов).
- Редизайн не делаем, только функциональные улучшения.
